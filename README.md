# simple_crud_api_go (WIP)

This is a simple CRUD API written in Go using the [gorilla/mux](https://github.com/gorilla/mux) package

| Endpoint     | Method |
| ------------ | ------ |
| /movies      | GET    |
| /movies/{id} | GET    |
| /movies      | POST   |
| /movies/{id} | PUT    |
| /movies/{id} | DELETE |

## Running the program

```console
go run main.go
```
